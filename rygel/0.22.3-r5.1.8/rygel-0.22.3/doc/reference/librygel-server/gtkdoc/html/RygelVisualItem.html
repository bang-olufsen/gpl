Property Details

The “color-depth” property

  “color-depth”              gint

The number of bits per pixel used to represent the video or image resource. A value of -1 means that the color depth is unknown and will not, or did not, appear in DIDL-Lite XML.

Flags: Read / Write

Default value: 0
